Filetype
=========

###Filetype ist einfach zu benutzen.

##BEISPIEL
```php
include "./src/type.php";
$pfad = "../../datei.docx";
$type = dateityp($pfad);
print_r($type);
```

Dieses Beispiel gibt zurück:

```php
Array ( [end] => docx [typ] => word [icon] => fa-file-word-o )
```

##INSTALLATION
Kopiere die Datei __type.php__ in einen beliebigen Ordner.

Fertig.
